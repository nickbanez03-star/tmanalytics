🔒 FACILITY PIN

Unlocks the iPad control app on a shared device. Optional — skip it and the app opens straight through. One per iPad.

🔑 PLAYER PIN

A quick 4-digit code to start a session as a given pitcher. Optional per player. Not the same as their portal password.

✉️ PORTAL LOGIN

The pitcher's own email + password for the remote portal. Linked to their facility profile by email, so sessions appear automatically.

🤖 WHAT'S AUTOMATIC

Ball-flight + pitch type capture themselves off the Rapsodo sniff. The only optional human taps: fixing a mis-classified pitch, and logging hitter outcomes during Live ABs.
